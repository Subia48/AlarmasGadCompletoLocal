# Manual de Usuario – Plataforma Web  
**Sistema Alarma Smart**

Versión: 1.1  
Última actualización: 2026  



## 1. Introducción

La plataforma web del sistema **Alarma Smart** permite la administración,
monitoreo y control de alarmas, usuarios y códigos de emergencia desde
un navegador web.

Este manual describe las funciones disponibles para los usuarios con
permisos de administrador.



## 2. Requisitos del sistema

- Navegador web actualizado (Chrome, Edge o Firefox)
- Conexión a Internet
- Credenciales de acceso válidas



## 3. Inicio de sesión

Al acceder a la plataforma web, el usuario visualizará la pantalla de inicio
de sesión.

### Campos disponibles
- **Correo electrónico**
- **Contraseña**

### Botones
- **Ingresar:** Accede al sistema.
- **Crear cuenta:** Redirige al formulario de registro.

![Inicio de sesión web](/img/im2/log1.jpg)



## 4. Registro de Usuario

Permite crear una nueva cuenta dentro del sistema.

### Datos del usuario
- Cédula
- Nombre completo
- Correo electrónico
- Teléfono
- Contraseña

### Contacto de emergencia
- Nombre del contacto
- Teléfono del contacto

### Botones
- **Registrarse:** Guarda la información.
- **Volver al login:** Regresa a la pantalla inicial.

![Gestión de alarmas](/img/im2/reg.jpg)

![Gestión de alarmas](/img/im2/reg2.jpg)



## 5. Panel principal – Gestión de Alarmas

Una vez iniciada la sesión, el sistema muestra el panel principal con un mapa
interactivo.

### Funciones disponibles
- Visualización de alarmas en el mapa
- Botón **Crear nueva alarma**

![Gestión de alarmas](/img/im2/me.jpg)





## 6. Creación de nueva alarma

Al presionar el botón **Crear nueva alarma**, se despliega un formulario modal.

### Campos del formulario
- Nombre de la alarma (opcional)
- Dirección de referencia
- Latitud
- Longitud
- ID del dispositivo

### Botones
- **Crear:** Registra la alarma.
- **Cancelar:** Cierra el formulario sin guardar.

![Nueva alarma](/img/im2/ca.jpg)



## 7. Módulo de Alertas

Permite generar alertas de tipo SOS de forma manual.

### Campos disponibles
- Latitud
- Longitud

### Botón
- **Enviar SOS:** Envía la alerta y la registra en el sistema.

![Alertas SOS](/img/im2/alert.jpg)



## 8. Gestión de Usuarios

Este módulo permite administrar los usuarios registrados en el sistema.

### Funciones disponibles
- Listado de usuarios
- Búsqueda de usuarios
- Cambio de rol (Usuario, Admin, Cuerpo SOS)
- Eliminación de usuarios

![Gestión de usuarios](/img/im2/edi.jpg)



## 9. Creación de nuevo usuario

Mediante el botón **Crear usuario**, se abre un formulario modal.

### Campos
- Cédula
- Nombre
- Email
- Contraseña
- Rol

### Botones
- **Guardar**
- **Cancelar**

![Nuevo usuario](/img/im2/edi2.jpg)


## 10. Códigos de Emergencia

Permite administrar códigos especiales para situaciones de emergencia.

### Crear nuevo código
- Código
- Descripción
- Botón **Guardar**

### Acciones disponibles
- Editar código
- Eliminar código

![Códigos de emergencia](/img/im2/cod.jpg)



## 11. Cierre de sesión

Para cerrar sesión, el usuario debe presionar el botón **Cerrar sesión**
ubicado en la parte superior derecha del sistema.



## 12. Consideraciones finales

La plataforma web de Alarma Smart centraliza la gestión del sistema,
permitiendo una administración eficiente, segura y en tiempo real.
